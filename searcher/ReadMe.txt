Installation instructions;

1. Navigate to the same directory as searcher.txt and requirements.txt.
run 
>>> pip install -r requirements.txt

2. Open a Python Terminal
run 
>>> import nltk
>>> nltk.download()

3. Extract extracted.zip. 
(To make check if you've done this correctly, you should have a folder called "extracted" with a lot of text files inside.)

4. Run searcher.py!

Et viola!
Have fun
